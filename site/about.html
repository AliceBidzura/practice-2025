О проекте

Цель проекта — автоматизация повседневных задач преподавателя: ведение журнала, создание расписания, проверка домашних заданий и формирование аналитики по ученикам. Мы используем Python, Flask, SQLite и OpenAI API для реализации ИИ-подсказок.

Ассистент будет доступен в виде веб-приложения с удобным интерфейсом и возможностью авторизации для преподавателей разных предметов.
